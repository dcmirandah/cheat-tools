# cheat-tools

This is a collection of scripts and configuration files designed to boost my productivity and streamline my environment setup.

## Features

- Handy shell scripts.
- Automated deployment of shell configuration (.zshrc for macOS, .bashrc for Linux).
- Custom aliases and utilities for git, Terraform, Ansible, and more.

## Quick Start

1. **Clone the repository:**

  ```bash
  git clone https://github.com/dcmirandah/cheat-tools.git
  cd cheat-tools
  ```

1. **Deploy tools and config:** This will copy the scripts and the appropriate shell config to your home directory.

  ```bash
  ./deploy.sh
  ```

## Directory Structure

- `configurations/` — Shell configuration files (.zshrc, .bashrc) and aliases.
- `tools/` — Handy shell scripts.
- `deploy.sh` — Deployment script for installing tools and configs.
